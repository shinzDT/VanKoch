# Koch Snowflake using WebGL

## Mô tả
Chương trình vẽ bông tuyết Koch sử dụng WebGL và thuật toán đệ quy.

## Cách chạy
1. Mở file index.html bằng trình duyệt
hoặc
2. Dùng Live Server trong VS Code

## Thuật toán
Thuật toán Koch chia mỗi đoạn thẳng thành 3 phần bằng nhau,
sau đó dựng tam giác đều ở đoạn giữa.
Quá trình được lặp lại đệ quy theo level.

## Tham số
- level: số lần đệ quy với min là 0, max là 6
